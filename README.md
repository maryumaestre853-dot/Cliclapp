# Ciclapp
CICLAPP es una aplicación para ciclistas urbanos que busca facilitar e incentivar la movilidad sostenible en Bogotá, mediante herramientas para consultar rutas seguras, ubicar puntos de servicio facilmente , registrar y dar un seguimiento a sus recorridos, y gestionar información del uso de la bicicleta como medio de transporte, para facilitar el estilo de vida de un ciclista y tenga mas comodidad al momento de hacer sus recorridos.

Lo que implementamos en este proyecto fue Google cloud dentro se encuentra las api de Google maps, y nuestros lenguajes de programacion fueron python, javaScript, React Native,  biblioteca node.
